# F.CSB305 — Программчлалын хэлний зарчмууд

## Лабораторийн ажил 1

## 1. Ажлын зорилго

Энэ лабораторийн ажлын зорилго нь энгийн арифметик илэрхийллийг задлан шинжилж, операторын давуу эрэмбийг зөв мөрдөн боддог жижиг интерпретатор хэрэгжүүлэх юм. Үүнд:

- Илэрхийллийг токенчлох (tokenize)
- Рекурсив задлагч (recursive descent parser) ашиглах
- Алдааны нөхцөлүүдийг боловсруулж, зөв мессеж өгөх
- Юнит тестээр найдвартай байдлыг баталгаажуулах

## 2. Хэрэгжүүлсэн боломжууд

Интерпретатор дараах боломжуудтай:

- Бинар операторууд: `+`, `-`, `*`, `/`
- Хашилт: `(`, `)`
- Олон оронтой бүхэл тоонууд (жишээ нь `100`, `2026`)
- Цагаан зайг үл тооцох (`" 3   +  5 "`)
- Операторын давуу эрэмбэ:
  - `*`, `/` нь `+`, `-`-аас түрүүлж бодогдоно
- Хашилтаар бодолтын дарааллыг өөрчлөх боломж
- Тэгээр хуваах, буруу тэмдэгт, хаагдаагүй хашилт зэрэг алдааг илрүүлэх

## 3. Ашигласан дүрэм (Grammar)

Дараах дүрмээр парсинг хийсэн:

```text
expr   -> term   { ('+' | '-') term }
term   -> factor { ('*' | '/') factor }
factor -> NUMBER | '(' expr ')'
```

Энэ бүтэц нь операторын давуу эрэмбийг автоматаар зөв хэрэгжүүлдэг.

## 4. Файлын бүтэц

- `interpreter.py` — үндсэн интерпретатор (tokenizer, parser, interpret, main)
- `test_interpreter.py` — `unittest`-ийн тестийн багц
- `result/` — ажлын үр дүн хадгалах хавтас (шаардлагатай бол)

## 5. Ажиллуулах заавар

### 5.1 Интерпретатор ажиллуулах

```bash
python3 interpreter.py
```

Жишээ:

```text
Илэрхийлэл оруулна уу: (4 * 8 + 3 - 1) - 7 * (9 / 2)
Үр дүн: 2.5
```

### 5.2 Тест ажиллуулах

```bash
python3 test_interpreter.py
```

## 6. Тестийн хамрах хүрээ

Тестүүдээр дараах тохиолдлуудыг шалгасан:

- Нэмэх, хасах, үржих, хуваах
- Давуу эрэмбэ (`2 + 3 * 4` гэх мэт)
- Хашилттай болон nested хашилттай илэрхийлэл
- Олон оронтой тоо
- Илүүдэл зай
- Алдааны тохиолдлууд:
  - Тэгээр хуваах
  - Танигдаагүй тэмдэгт
  - Хаагдаагүй хашилт

Нийт **20 тест** амжилттай ажиллаж байна.

## 7. Жишээ оролт/гаралт

| Оролт               | Гаралт              |
| ------------------- | ------------------- |
| `3 + 5`             | `8`                 |
| `10 - 6 / 2`        | `7`                 |
| `(2 + 3) * (4 - 1)` | `15`                |
| `5 / 0`             | `ZeroDivisionError` |
| `3 @ 5`             | `ValueError`        |

## 8. Дүгнэлт

Энэхүү лабораторийн ажлаар арифметик илэрхийлэл интерпретаторын үндсэн ойлголтуудыг (tokenization, parsing, evaluation) амжилттай хэрэгжүүллээ. Програм нь энгийн математик илэрхийллийг зөв бодож, нийтлэг алдаануудыг зохих ёсоор барьж чаддаг бөгөөд юнит тестүүдээр баталгаажсан.
